  1. Что такое CSS? CSS (Cascading Style Sheets) — это язык стилей, который используется для оформления веб-страниц. Он определяет, как элементы HTML будут отображаться в браузере. Основное преимущество CSS — разделение структуры и дизайна, что упрощает разработку и поддержку веб-проектов.
  2. Подключение CSS к HTML Существует три способа подключения CSS: встроенный стиль через атрибут style в тегах HTML, внутренние стили в блоке <style> внутри HTML-документа и внешний файл CSS, подключаемый через <link>. Использование внешнего файла считается лучшей практикой, так как это облегчает управление стилями.
  3. Основы селекторов и свойств Селекторы в CSS позволяют выбирать элементы HTML для применения стилей. Например, селектор p выбирает все параграфы, а селектор .class — элементы с определённым классом. Свойства CSS определяют, какие стили применяются, например, color задаёт цвет текста, а margin отвечает за отступы.
